设计稿说明 - Raincat

目标：清晰、极简、卡片式项目展示页；优先移动端适配；此文件用于在 Canva/Figma 中复现设计。

配色：
- 主色：#3B82F6
- 背景：#F7FBFF
- 文本：#0B1220
- 次要文本：#6B7280

字体：Inter / Noto Sans SC（标题 600，正文字重 400）

布局：12 列网格，容器宽度 1100px，左右内边距 24px；卡片圆角 10px，阴影轻量

组件：
- 顶部导航（logo + 链接）
- Hero（H1、lead、按钮）
- 项目卡片网格（缩略图占位、标题、描述）
- Footer（版权 + 备案）

导出指南：
- 画板：Desktop 1440×900 / Tablet 1024×768 / Mobile 375×812
- 导出 PNG 2x，用于演示
- logo 导出 48×48 与 96×96

注意：用户要求“无需图片”，预览将使用占位背景代替实际图片。
